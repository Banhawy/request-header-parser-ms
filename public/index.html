API Request Header Parser Microservice

This is part of FreeCodeCamp's API Project challenges.

User stories:

  1. I can get the IP address, language and operating system for my browser.

Example usage:

https://adham-request-header-parser-ms.herokuapp.com/api/whoami

Example Output:

{ "ipaddress": "107.2.92.3", "language": "en-US", "software": "Windows NT 10.0; Win64; x64" }
